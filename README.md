# Enochian Workbench

This is a hobby project with the goal of learning React (and later backend)
for an application that displays various versions of the four Enochian Tablets
as used in Golden Dawn (and other) forms of ceremonial magic.

It basically presents a set of four 12x23 grids and allows the user to select sets of cells, and to save and retrieve
various 'interesting' sets of cells.
